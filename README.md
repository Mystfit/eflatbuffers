# Eflatbuffers

## features from flatc

* tables
* scalars
* strings
* vectors
* unions
* enums
* defaults
* json to fb
* fb to json
* file identifier
* random access

## features not in flatc

* vectors of enums

## deviation from flatbuffers

* default values are written to json

## pending features

* validate file identifiers
* shared strings
* shared vtables

## not planned

* includes
* alignment
* additional attributes
* structs
