# Eflatbuffers

## features from flatc

* tables
* scalars
* strings
* vectors
* unions
* json -> fb
* fb   -> json
* enumerables
* file identifier

## features not in flatc

* nested vectors
* vectors of enums

## deviation from flatbuffers

* default values are written to json

## pending features

* random access
* validate file identifiers
* shared strings
* shared vtables

## not planned

* includes
* alignment
* additional attributes
* structs
